# Smart Meter

## Requirements

- Node.js Version >= 18
- npm Version >= 10

This project uses Next.js 13.5.6.

## Start application

`npm install`
`npm run build`
`npm run start`

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Docker

`npm install`
`npm run build`
`docker build -t smart-meter-ui .`
`docker run -p 3000:3000 smart-meter-ui`
